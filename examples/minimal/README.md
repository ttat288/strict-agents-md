# Minimal Example

For the smallest setup, copy the root [`AGENTS.md`](../../AGENTS.md) and [`PROJECT.md`](../../PROJECT.md) into your project root.

No tool-specific wrapper is required.

Use this when one portable behavior file plus one small project-decision file is enough:

```txt
/
|-- AGENTS.md
`-- PROJECT.md
```

Add `knowledge/` later only if stable domain context becomes too large for `PROJECT.md`.
